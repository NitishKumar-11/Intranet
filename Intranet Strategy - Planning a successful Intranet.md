# Actual Planning and Implementation

### For an intranet, a typical strategy would include the following elements:
1. Vision
    
2. Mission
  
3. Target audience definition / segmentation
   
4. Governance
    
5. Goals
 
6. Objectives
   
7. Action plans

8. KPIs (or CSIs)
